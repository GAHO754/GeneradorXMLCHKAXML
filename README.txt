# Generador XML desde CHK

Página web local para convertir archivos .CHK a XML con estructura interna tipo factura.

## Archivos

- index.html
- styles.css
- app.js

## Uso

1. Abre index.html en el navegador.
2. Sube un archivo .CHK.
3. Revisa la vista previa.
4. Presiona Generar XML.
5. Descarga el XML.

## Nota importante

Este XML NO es un CFDI timbrado ni válido ante SAT. Es una estructura interna de factura para pruebas o integración.
